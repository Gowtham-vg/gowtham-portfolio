# Gowtham B — Developer Portfolio

Personal portfolio website built from scratch showcasing my projects, skills, and experience as a Python & Flask developer.

🔗 **Live Site:** [gowtham-vg.github.io/gowtham-portfolio](https://gowtham-vg.github.io/gowtham-portfolio)

---

## Built With

- HTML5, CSS3, JavaScript (Vanilla)
- CSS custom properties for theming
- Intersection Observer API for scroll animations
- Canvas API for particle background
- Google Fonts (Syne + DM Sans)
- Deployed via GitHub Pages

## Features

- Responsive design across all screen sizes
- Animated skill bars and scroll reveal effects
- Interactive particle background
- Custom cursor
- Project cards with GitHub links
- Contact section with live links

## Sections

- Hero with availability status
- Technical skills with proficiency bars
- Work experience timeline
- Projects with GitHub links
- Education and certifications
- Contact
